# RCO IT Help Portal (Demo)

> **This repo is for demo purposes only.** Firebase authentication is replaced with a mock login, and form submissions are stored in-memory only.

**View the demo site:** [hhowell116.github.io/Demo-RCO-Help-Site](https://hhowell116.github.io/Demo-RCO-Help-Site/)

**View the real project code:** [github.com/hhowell116/IT-Help-Site](https://github.com/hhowell116/IT-Help-Site)

---

An internal IT portal for Rowe Casa Organics staff to submit technology surveys, onboarding/offboarding requests, and equipment requests. Features Google SSO restricted to @rowecasaorganics.com accounts, Cloud Firestore-backed submissions, Google Sheets integration via Apps Script, and dark mode support.
